## State Before Style

Before adding polish, I like to map the UI states:

- idle
- hover or focus
- pressed
- loading
- success or error

<BlogCallout tone="mint">
  A cute button is still a system. It needs clear states before it needs
  sparkles.
</BlogCallout>

<ProjectNote title="Roblox note">
  Luau modules help keep UI state separate from gameplay logic, which makes
  iteration less risky.
</ProjectNote>
